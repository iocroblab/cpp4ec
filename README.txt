em4cpp
------


Ethercat Master for C++.

The main objective of this library is to encapsulate the soem [1] library
in a class to use in a C++ program. The library is inspired in the philosofy
of the soem master [2], but decoupling from the orocos framework.

It also could use Xenomai as Realtime interface.






[1] http://soem.berlios.de
[2] http://git.mech.kuleuven.be/robotics/soem.git
